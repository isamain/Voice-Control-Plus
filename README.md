# Voice-Control-Plus
En / What does it do?

it allows you to automatically disconnect, mute, deafen the person you choose from the audio channel at all times.

How to use?

After determining the target user, we right-click and click the (disconnect user/mute or deafen) button and disconnect it ourselves. That's it, then the plugin will do its job and constantly mute deafen or disconnect that user.

How to close?

We click the button again in the menu of the user we selected and it is closed, that's it.

Have Fun :)
